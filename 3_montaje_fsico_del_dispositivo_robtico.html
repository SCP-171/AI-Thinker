3. Montaje físico del dispositivo robótico

Instrucciones paso a paso

Instrucciones paso a paso

PROMPT: Proporciona una guía detallada con imágenes o diagramas para facilitar el ensamblaje.​

Materiales necesarios:
-1 placa ESP32-CAM
-1 protoboard (opcional)
-Cables jumper hembra-hembra (opcional)
-1 sensor de movimiento PIR (opcional)
-1 tarjeta microSD (opcional)
-1 cable USB para el módulo FTDI

Paso 1: Conexión para programar la ESP32-CAM
Configuración del módulo FTDI: asegúrate de que está en modo 5V, no en 3.3V.

Paso 2: Programación desde el ordenador
-Abre el Arduino IDE.
-Instala el soporte para placas ESP32 (si aún no lo tienes).
-En el menú Herramientas, selecciona:

1.Placa: AI Thinker ESP32-CAM
2.Puerto: el que detecte el FTDI
3.Abre el ejemplo: Archivo > Ejemplos > ESP32 > Camera > CameraWebServer.
4.Configura tu red WiFi en el código.
5.Sube el código a la placa. 

Paso 3: Alimentación y prueba
Una vez cargado el programa:

1.Abre el monitor serie del Arduino IDE a 115200 baudios para ver la IP.
2.Copia esa IP en el navegador web desde un dispositivo conectado a la misma red WiFi.
3.Deberías ver el panel de control de la cámara.

Paso 4: Integración de sensores (opcional)
Puedes conectar un sensor PIR al pin GPIO13, por ejemplo, y modificar el código para que la cámara se active solo al detectar movimiento. También puedes guardar imágenes o vídeo en una tarjeta microSD, conectada a la ranura de la placa.

Paso 5: Presentación y montaje final
Se recomienda montar todos los componentes en una base (como cartón pluma o una caja) y proteger la cámara con una carcasa si es posible. Cada grupo deberá presentar:

-El dispositivo funcionando.
-Una explicación de cómo lo han montado y programado.
-Una propuesta de aplicación práctica en su entorno (instituto, casa, tienda, etc.).
