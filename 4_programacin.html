4. Programación

Entorno de programación

Entorno de programación

PROMPT: Explica el Entorno de Programación: Arduino IDE

1. Arduino IDE (Recomendado)
¿Qué es?
Es un entorno de desarrollo gratuito y fácil de usar que permite programar placas como Arduino, ESP8266 y ESP32 utilizando un lenguaje basado en C/C++ simplificado.

¿Por qué usarlo?
-Interfaz sencilla.
-Mucho material didáctico y ejemplos.
-Compatible con ESP32-CAM.
-Permite cargar código mediante el módulo FTDI.

¿Qué necesitas?
-Descargar el Arduino IDE desde:
https://www.arduino.cc/en/software
-Instalar el soporte para placas ESP32:

1.Abre el IDE y ve a: Archivo > Preferencias.
En “Gestor de URLs adicionales de tarjetas”, añade:

https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json

-Luego ve a: Herramientas > Placa > Gestor de placas y busca e instala esp32 by Espressif Systems.
Selecciona la placa AI Thinker ESP32-CAM desde el menú Herramientas > Placa.

Desafíos de programación

Desafíos de programación

PROMPT: Plantea retos de dificultad creciente para fomentar el pensamiento computacional.

Reto 1: Haz que la cámara funcione

-Subir el ejemplo básico CameraWebServer al ESP32-CAM.
-Ver la imagen de la cámara desde el navegador.

Reto 2: Cambia la resolución de imagen

-Modificar el código para cambiar la calidad o tamaño de imagen (por ejemplo, de VGA a SVGA).
-Observar las diferencias en la fluidez o calidad.

Nivel 2: Intermedio (Modificación y aplicación lógica)
Objetivo: Comenzar a modificar el comportamiento del sistema según condiciones simples.

Reto 3: Detectar movimiento con un sensor PIR

-Conectar un sensor de movimiento.
-Modificar el código para que solo se active la cámara cuando se detecte movimiento.
